Simple Calculator

The Simple Calculator is a beginner-friendly web application built using Vanilla JavaScript. It performs basic arithmetic operations and demonstrates core JavaScript concepts in a clean and functional interface.

This project is designed to reinforce the foundational concepts of JavaScript such as:

  • 🔹 Variables – for storing user inputs and results
  • 🔹 Data Types – handling numbers and operations
  • 🔹 Control Structures – managing calculator logic
  • 🔹 Loops – iterating through operations (where needed)
  • 🔹 Functions – for reusable and modular code blocks

The calculator supports operations like addition, subtraction, multiplication, and division. It's fully responsive and styled with minimal CSS for a clean layout.
