# Vanilla JS Mastery

## 📖 About This Repository
This repository is a structured collection of JavaScript projects, grouped by concepts from basics to advanced levels. Each folder focuses on a specific topic, ensuring progressive learning and mastery of Vanilla JavaScript.

## 📂 Project Structure
- **01_Getting_Started/** → Introduction to JavaScript, setup, and basic syntax
- **02_Basics/** → Variables, data types, operators, conditionals, loops, functions
- **03_Working_With_Browser/** → The DOM, events, forms, and user interaction
- **04_Advanced_JS/** → Objects, classes, prototypes, ES6+ features, modules
- **05_Asynchronous_JS/** → Callbacks, Promises, Async/Await, AJAX, Fetch API
- **06_Error_Handling_Debugging/** → Common errors, debugging tools, best practices
- **07_Modern_JS_Tools/** → Babel, Webpack, NPM, modern development workflows
- **08_JS_Projects/** → Practical projects implementing core JavaScript concepts

## 🎯 Learning Objectives
- Strengthen core JavaScript concepts
- Build interactive web applications without frameworks
- Understand modern JavaScript features
- Improve problem-solving and debugging skills

## 🚀 How to Use
1. Clone the repository:  
   ```sh
   git clone https://github.com/your-username/vanilla-js-mastery.git
   ```
2. Navigate to a project folder:  
   ```sh
   cd vanilla-js-mastery/02_DOM_Manipulation
   ```
3. Open `index.html` in your browser and explore the code.
